# Lost Frogs

top-down puzzle game.
navigate a frog to it's destination,
then rewind time and repeat for the remaining frogs.

![green-frog](godot/assets/frogs/green-frog.png)

## Tools
### godot
engine and editor
### libresprite
asset generation
### obsidian
notes, docs, issues
